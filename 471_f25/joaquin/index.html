Project Overview

Heaven & Hell Minigame is an OpenGL-based action game in which the player begins in a peaceful heaven setting, ascends a staircase, and enters a portal that transforms the world into a fiery hell arena filled with demons.

Your mission is to defeat the smaller flocking demons before facing the main demon, using a throwable magical staff. This project integrates lighting, shading, hierarchical animation, particle effects, collision detection, flocking AI, and real-time audio.

Graphics Technologies Used
  • Blinn–Phong shading
  • Material system (ambient, diffuse, specular)
  • Texture-mapped shaders + dynamic skyboxes
  • Particle effects (fire, smoke, ash)
  • Boid system (leader–follower demon flocking)
  • Hierarchical modeling for animated demons
  • Spline-based cinematic camera
  • Projectile system + hit detection
  • FMOD audio system (SFX + music)
  • WASD + mouse-look free camera
