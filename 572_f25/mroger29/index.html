Description and Methods

Overview

There is the local motion of the branches and leaves. This effects the bend amount and sway speed of the tree. This motion is based on a sine wave with noise added to it so the motion is not completely uniform.

There is a jitter effect on the leaves only. It contributes to the Extra leaf movement The lighter elements, like the leaves, are more affected by wind so they have another level of motion added to them.

This project was created with Blender’s Python. Adding a frame driver is easy with Blender’s normal tools, but with scripting I had to find the path to specific data in order to access and modify it. This took multiple deep dives onto the documents page and console. I wrote a delete function so I could run that each time instead of going to each element and resetting them to have no modifiers.
